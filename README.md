# MapAction 2026-FSM-001 MDR holding site

Minimal website containing maps produced for the Cyclone Sinlaku response in the Federated States of Micronesia.

Used as a temporary workaround for issues with MapAction Maps and Data Repository (MDR) to ensure maps for the current response are available.

For MA members, see https://mapaction.atlassian.net/browse/SDO-37 for more information.
